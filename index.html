Croissance d'une bactérie

Une culture bactérienne contient u₀ = 50 bactéries au démarrage. Le nombre de bactéries double chaque heure (donc q = 2).
Construis le diagramme en bâtons représentant uₙ pour n = 0, 1, 2, 3, 4 et 5. Fais glisser le sommet doré de chaque bâton à la bonne hauteur.

u₀ = 50, puis chaque terme se calcule en multipliant le précédent par 2. u₁ = 100, u₂ = 200, etc.

Exercice 03 Lire et identifier

Quelle suite reconnais-tu ?

Le graphique ci-dessous représente les six premiers termes d'une suite géométrique. Lis les valeurs sur le graphique et réponds aux questions. Tu peux passer la souris (ou le doigt) sur un bâton pour afficher sa valeur exacte.
Pose la souris sur un bâton pour voir sa valeur · Lis les hauteurs sur l'axe vertical
Quelle est la valeur de u₀ ?
Quelle est la raison q ?
Cette suite est-elle croissante ou décroissante ?
Pour trouver q, calcule le rapport entre deux termes consécutifs : q = u₁ ÷ u₀. Pour la nature, regarde si les bâtons montent ou descendent.

Lecture graphique : u₀ = 80, u₁ = 60, u₂ = 45, u₃ ≈ 33,75, u₄ ≈ 25,3, u₅ ≈ 19.

On calcule la raison : q = u₁ ÷ u₀ = 60 ÷ 80 = 0,75.

Vérification : u₂ ÷ u₁ = 45 ÷ 60 = 0,75 ✓ — le rapport est bien constant.

Comme 0 < q < 1, la suite est décroissante. C'est confirmé par le graphique : les bâtons diminuent.

✓ u₀ = 80 · q = 0,75 · suite décroissante.

Exercice 04 ★ Comparer deux scénarios

Deux placements bancaires

Deux amis placent chacun 1 000 € dans des banques différentes :
  • Anaïs place chez la banque A : taux fixe de 5 % par an (qₐ = 1,05).
  • Bilal place chez la banque B : taux fixe de 3 % par an (q_B = 1,03), mais reçoit en plus une prime de bienvenue de 100 € à l'ouverture (donc u₀ᴮ = 1 100).
1. Construis le diagramme en bâtons des deux placements pour n = 0 à 5 (les bâtons bleus pour Anaïs, les bâtons rouges pour Bilal).
2. Lis graphiquement à partir de quelle année le placement d'Anaïs dépasse celui de Bilal.

Ce que retenir

Un graphique permet de comparer rapidement deux scénarios sans calculer toutes les valeurs. Au CCF, on te demandera souvent de lire un croisement, un seuil ou de comparer deux courbes : sois rigoureux dans la lecture des hauteurs.
